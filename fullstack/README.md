# Como rodar a API em produção:

## Requisitos mínimos:
- Node 18.18.0;

## Como executar a aplicação:
- Instale as dependências
  ```
  npm i
  ```

- Crie o build da aplicação
  ```
  npm run build
  ```

- Inicie a aplicação
  ```
  npm run start

- Para rodar os testes da aplicação:
```
  sudo apt intall jest

  jest
  ```
